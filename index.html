Composants

Le Raspberry Pi 2 Model B

Il s'agit du dernier modèle vendu : possèdant 1 Gb de mémoire vive, un processurs quad-core ARM Cortex-A7 et 4 port usb.
Il est suffisament puissant pour décoder des vidéos Full HD.

Pour les spécifications exactes, vous pouvez consulter la page du Raspberry Pi 2 Model B sur le site dédié à la petite carte.

L'alimentation

Pour fonctionner, votre Raspberry Pi à besoin d'une alimentation. Un chargeur de mobile au format micro-usb suffit.

La carte micro SD

Le Raspberry Pi a besoin d'une carte micro SD (jusqu'a 32 Gb), sur laquelle est installée le système d'exploitation, pour fonctionner.

Le XID se fera un honneur de vous aider a installer une des nombreuses distributions disponnible.

Un câble HDMI

Vous avez envie de bénéficiez de toute la puissance du Raspberry Pi pour décoder vos Blu-ray ?

Il vous suffit de le racorder sur votre écran ou téléviseur HD avec un câble HDMI, et hop à vous les films de bonne qualité !
