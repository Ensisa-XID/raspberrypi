Composants

Le Raspberry Pi

Il s'agit du dernier modèle vendu : le modèle B possèdant 512 Mb de mémoire vive.
Son circuit graphique lui permet notament de décoder les flux vidéos Full HD.

Pour les spécifications exactes, vous pouvez consulter la page Wikipédia qui lui est consacré.

L'alimentation

Pour fonctionner, votre Raspberry Pi a besoin d'une alimentation. Un simple câble micro-USB lui permet de fonctionner.

Pour plus de commodités, nous vous avons choisi un câble µUSB disposant d'un adaptateur secteur.

Vous pouvez voir à quoi ressemble cette alimentation.

Le boitier

Trois couleurs du boitier spécialement conçu pour le Raspberry Pi sont disponibles à la commande :

  • boitier transparent,
  • boitier blanc,
  • ou boitier noir.

Vous pouvez aussi vous amusez à fabriquer le vôtre, par exemple avec des Legos...

La carte SD

Le Raspberry Pi a besoin d'une carte SD, sur laquelle est installée le système d'exploitation, pour fonctionner.
Nous vous avons choisi une carte SDHC classe 6 de 4Go pour que vous puissiez y mettre le système d'exploitation.

Le XID se fera un honneur de vous installer une des distributions suivantes si vous le souhaitez :

  • Raspbian
  • ArchLinux ARM v6
  • OpenELEC v3

Le câble HDMI

Vous avez envie de bénéficiez de toute la puissance du Raspberry Pi pour décoder vos Blu-ray ?

Il vous suffit de le racorder sur votre écran ou téléviseur HD avec le câble HDMI, et hop à vous les films de bonne qualité !

Nous avons selectionné un câble HDMI de 1,8 mètres, avec connecteurs en or, compatible HDCP.
